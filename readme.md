# Implementation of Marching Cube Algorithm

A Terrain Editor written in **THREEJS** for demo of the marching cube algorithm based on this article (https://paulbourke.net/geometry/polygonise/).

# Live Demo
[Checout the Live Demo](https://marchingcube.netlify.app/)


# Run the Project
```
# Clone the repo
git clone https://github.com/kendad/MarchingCubeThreeJS.git
cd MarchingCubeThreeJS

# NPM install the required libraries
npm install

# Run using command
npm run
```
